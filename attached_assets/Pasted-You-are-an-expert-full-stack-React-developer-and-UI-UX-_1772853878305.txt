You are an expert full-stack React developer and UI/UX designer specializing in high-conversion e-commerce experiences with bold, editorial aesthetics. Build a complete, production-ready e-commerce website for a clothing brand called **Outlaw Dolls**.

---

**Brand Identity**

- **Tagline:** "Break Expectations and Steal the Spotlight."
- **Aesthetic:** Dark, edgy, baddie — nighttime energy, leather, moody editorial photography, powerful femininity. Every design decision should feel intentional and luxe.
- **Color palette:**
  - Background: `#000000` and `#111111`
  - Text/neutral: `#F5F5F0` (off-white)
  - Accent: Gold `#C9A84C` — use sparingly for maximum impact (CTAs, hover states, underlines, borders)
- **Typography:**
  - Headings: Cormorant Garamond or Playfair Display (import from Google Fonts)
  - Body: DM Sans
- **Logo:** Serif script "OUTLAW DOLLS" — render as styled text using the heading font, letter-spaced and uppercase

---

**Tech Stack**

- React (functional components + hooks)
- Tailwind CSS (configure custom colors and fonts in `tailwind.config.js`)
- React Router v6 for all navigation
- Context API for global cart state (`CartContext`)
- `products.js` mock data file with 6 placeholder items using Unsplash dark fashion image URLs

---

**File Structure to Generate**

Scaffold a clean, organized project:

```
/src
  /components     → Navbar, Footer, ProductCard, CartDrawer, Marquee
  /context        → CartContext.js
  /data           → products.js
  /pages          → Home, Shop, ProductDetail, Cart, Checkout, About, Contact
  /styles         → global.css (Tailwind base + custom animations)
App.jsx
tailwind.config.js
```

**Pages & Features**

**Homepage (`/`)**
- Full-screen hero with a dark editorial Unsplash background image, subtle dark overlay
- Animated headline using CSS keyframes or Framer Motion: "Break Expectations. Steal the Spotlight." — heading font, large, off-white
- Gold-accented CTA button ("Shop Now") linking to `/shop`
- Scrolling marquee below hero: `"Break Expectations · Steal the Spotlight · Outlaw Dolls ·"` — infinite loop, smooth CSS animation
- Featured products section (pull first 3 from `products.js`)

**Shop Page (`/shop`)**
- Responsive product grid (2 cols mobile, 3–4 cols desktop)
- Each `ProductCard`: dark card background (`#111`), product image with hover zoom (`transform scale-105 transition`), product name, price, and "Add to Cart" button with gold hover state
- Cart item count updates in Navbar on add

**Product Detail Page (`/product/:id`)**
- Large hero image (left) + product info (right) two-column layout
- Size selector (XS, S, M, L, XL) as styled toggle buttons
- "Add to Cart" button — full-width, gold on hover
- Short brand description: edgy, confident tone matching brand voice
- Related products row at bottom (2–3 other items from `products.js`)

**Cart (`/cart`)**
- Line items with image thumbnail, name, size, quantity stepper, remove button
- Order summary with subtotal
- "Proceed to Checkout" CTA

**Checkout (`/checkout`)**
- Minimal single-page form: name, email, address, card fields (static/non-functional)
- Order summary sidebar
- "Place Order" button — on click, clear cart and show a confirmation message

**About Page (`/about`)**
- Editorial two-column layout alternating text + dark fashion images
- Brand story copy in the Outlaw Dolls voice: rebellious, feminine, unapologetic
- Tagline displayed large as a section divider

**Contact Page (`/contact`)**
- Email contact form (name, email, message) — styled dark inputs with gold focus borders
- Social links: Instagram, TikTok, YouTube (use react-icons or SVG icons)

**Global UI Details**

- **Navbar:** Sticky, transparent-to-dark on scroll. Logo centered. Nav links ("Shop," "About," "Contact") on the right. Cart icon with live item count badge on the left. All links gold on hover.
- **Footer:** Newsletter email signup input + subscribe button. Social icons (IG, TikTok, YouTube). Copyright line: `© 2025 Outlaw Dolls. All rights reserved.`
- **Transitions:** Smooth fade-in on page load using CSS or Framer Motion. Scroll-reveal for section entries.
- **Mobile-first:** All layouts fully responsive. Hamburger menu on mobile.
- **Dark mode default:** No light mode toggle needed — this is the brand.
